Launch PhenologyMMS 1.3

Install the model and double click the desktop icon to launch the program.

 

The first screen to appear is the PhenologyMMS 1.3 Flash Screen or Welcome Screen which simply shows the name of the model and the authors.

 

 

It is displayed for about 2 seconds and then disappears allowing the Begin Setup screen to display.
